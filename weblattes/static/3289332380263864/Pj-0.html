1.   2007-2008. Ventura: Uma abordagem baseada plugins para a construção de interfaces gráficas com o usuário
Descrição: O desenvolvimento de ferramentas para a construção de aplicações é uma temática bastante discutida nos dias atuais. Diversos avanços a esse respeito se deram ao nível de interfaces gráficas. Entretanto, grande quantidade de avanços se focam na edição visual de interfaces. Contudo, essas ferramentas possuem algumas limitações e não são muito flexíveis para a construção de interfaces gráficas dinâmicas. Esse projeto tem o objetivo de desenvolver uma ferramenta que auxilem na construção de interfaces gráficas com o usuário. As tecnologias a serem utilizadas serão Java, XML, além da api Reflection.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Daniel Michelon De Carli - Coordenador / Marcos Cordeiro d'Ornellas - Integrante. 1.   2006-2006. Ferramenta para o rastreamento e visualização de programas Java usando JVMTI
Descrição: Este projeto tem como objetivo desenvolver uma ferramenta para a monitoração e rastreamento de programas Java para posterior visualização. Essa ferramenta deve ser de simples utilização, permitindo o rastreamento de programas Java sem a necessidade do desenvolvedor modificar o código da aplicação, ou da máquina virtual. Além disso, a ferramenta deve ser configurável, a permitir ao usuário escolher entre as diversas informações que podem ser registradas de forma a controlar o equilíbrio entre a quantidade de informação desejada e a interferência causada na execução da aplicação devido a esse registro... Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . Integrantes: Daniel Michelon De Carli - Integrante / Geovani Ricardo Wiedenhoft - Coordenador / Benhur de Oliveira Stein - Integrante. Financiador(es): Universidade Federal de Santa Maria - Bolsa.
Membro: Daniel Michelon De Carli.
2.   2006-2006. Sistema de Gestão de Certificado Usando PHP 5 Orientado a Objetos em MySQL
Descrição: Ferramenta que possibilita o cadastro de encontros e eventos, alunos e cursos, bem como relacionar alunos à participação em tais eventos e encontros, afim de centralizar e automatizar a geração de certificados. A ferramenta é capaz de realizar a confecção de certificados dos eventos e encontros com seus devidos participantes, bem como disponibilizar uma página web para que qualquer participante de algum evento cadastrado possa verificar os certificados que já foram gerados e/ou entregues.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (4) / Mestrado acadêmico: (1) . Integrantes: Daniel Michelon De Carli - Integrante / Ronaldo Canofre Mariano dos Santos - Integrante / Caroline Fighera Vicentini - Integrante / Fernando Bevilacqua - Coordenador / Oni Reasilvia de Almeida Oliveira Sichonany - Integrante.
Membro: Daniel Michelon De Carli.
